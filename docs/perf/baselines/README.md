# Baseline Files

- `rfxjit_microkernels_cpu.json`: tracked CPU baseline used by CI soft perf gate.
- `rfxjit_microkernels_cuda.json`: optional tracked CUDA baseline (future CI GPU runners).
- `rfxjit_microkernels_metal.json`: optional tracked Metal baseline (future CI macOS runners).

Local pre-push perf baselines are stored in `.rfx/perf-baselines/`.
