# rfx

A robotics framework with a Rust core and Python bindings.

## Features

- Control systems (PID controllers, state machines, control loops)
- Math utilities (quaternions, transforms, filters)
- Hardware abstractions (Unitree Go2 support)
- Simulation backends
- RL training and neural policy support
- Agent control and skills system

## Installation

```bash
pip install git+https://github.com/quantbagel/rfx.git
```

## Development

Requires Rust and Python 3.8+.

```bash
# Install in development mode
pip install -e .
```
